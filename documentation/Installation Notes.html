eloipool + bitcoind Ubuntu 12.04 Private Server Installation Notes

Last updated on March 31, 2013

Notes for Ubuntu 12.04

Expand the archive and edit the config files. Then run the 'Double click to start install.sh' followed by scripts/startpool.sh to start the server.

Tips:

  • Edit the bitcoin.conf in eloipool-server/bitcoindata and change the RPC password to something unique and secure. (Use: $ apg)
  • Edit eloipool/config.py and update the rpcuser and password to match bitcoin.conf.
  • Remember to update the TrackerAddr in eloipool/config.py to a valid wallet address.
  • Think about setting the log backupCount in eloipool/config.py to 1.
  • Run eloipool-server/scripts/startpool.sh to start the server. It can be run when the user logs in or from cron.
  • The pool username and password can be anything.
  • If you're using a Linux Disribution other than Ubuntu, comment out the lines that have the command 'sed' in them, remove the double click script and ubuntu-install.sh.
  • For more information check https://bitcointalk.org/index.php?topic=158105.0.
